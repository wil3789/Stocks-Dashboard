Welcome to the Stocks Dashbaord

Get historical performance for any stock of your choice, if it is on a major American exchange. Here are the instructions:

  • 1. Click on 'Input Form' in the navigation bar
  • 2. Input your AlphaVantage API key
  • 3. Input the ticker of the stock you want to display
  • 4. Select a benchmark from the drop-down menu
  • 5. Input the number of trading days you would like to show in the chart
  • 6. Click the 'Submit' button

Note: This page is in development. More functionality will be added soon.
